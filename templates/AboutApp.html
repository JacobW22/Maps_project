Shop Finder is an application that allows you to search and display all the stores across Poland. The application is based on data from https://www.promoceny.pl/sklepy/, which is collected using Web Scraping and kept in the database. There is also a free API that sends the collected information about stores.
Search by shop name

Using the select menu, you can choose any store from the list. Then the application will display all its addresses in the country.

Shop Finder v.1.0

Search by city name

Using the select menu, you can choose any city from the list. Then the application will display all the shops in the city.

Shop Finder v.1.0

Non included addresses

Due to the use of an inaccurate geocoder, some addresses may not be processed into coordinates. They will appear in the "Non-included" section.

Shop Finder v.1.0

Map properties

Addresses are displayed by heatmap and markers. We can choose between a view of all stores in the selected city or a view of all shop addresses.

Shop Finder v.1.0

Reset database

The Reset Database button deletes all data from the database and reloads it. While loading the data, you can use the data loaded so far. The loading itself takes a long time.
